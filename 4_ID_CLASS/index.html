ID e CLASS

ID

O ID funciona como um CPF. assim como só posso ter tem número de cada elemento só pode ter um valor de ID. O CPF é único, ou seja ninguém pode ter um CPF igual ao meu; Isso significa que o ID ta é único. então não é possivel repetir o mesmo valor duas vezes.

CLASS

O CLASS funciona de maneita oposta ao ID. Diferente do ID em que você so pode ter um valor, no CLASS você pode ter vários valores. Também diferente do ID, em que nao se pode repetir um valor, no CKASS você pode ultilizar o mesmo valor para vários elementos.
